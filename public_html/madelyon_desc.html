Compétences et Détails du Projet

Le projet

Le projet de créer un site web pour un restaurant fictif. se situe vitrine a pour but de donner une visibilité à un rentrant bar créer récemment

Technologies

Les technologies que j'ai surtout utilisées sont le HTML et le CSS parce que notre site demeure très simple.

Contexte

ce projet a été réaliser dans le crade d'une semaine de workshop créer par l'école dans le but créer une entreprise fictive pour quand puissent explorer plusieurs domaines et métiers comment comptable, agents immobilier et même développer

Compétences

Dans ce projet la compétence qui était la plus indispensable pour moi c'était le travail d'équipe car on était composé de groupe d'élever qui étudiait des métiers très différents
